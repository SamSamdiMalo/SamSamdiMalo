### [Arch Linux Minimalist Programming Environment](https://github.com/SamSamdiMalo/dotfiles-arch-bspwm)
- **The Achievement:** Revived a legacy 2007 hardware platform (Pentium Dual Core, ~3GB RAM) into a highly efficient, 20-second boot workstation.
- **The Stack:** Arch Linux, `bspwm`, `sxhkd`, `Alacritty`,`picom`,`hsetroot` and `lemonbar`.
- **Key Takeaway:** Achieved an idle memory consumption of just ~505 MiB, mastering low-level system configuration, hardware resource constraints, and dotfile deployment.
